# Number Guessing Game (Java)

## Project Description
This is a simple Java console-based Number Guessing Game.  
The program generates a secret number and the user must guess it.  
The program will guide the user by telling whether the guess is **too high**, **too low**, or **correct**.

This project is useful for beginners to understand:
- Java loops
- Conditional statements
- User input using Scanner
- Basic input validation

## Features
- User guesses a number between **1 and 100**
- Program gives hints:
  - Too High
  - Too Low
  - Correct Guess
- Handles invalid input (non-numeric values)
- Runs continuously until the correct number is guessed

## Technologies Used
- Java
- Scanner Class
- Console Input/Output
